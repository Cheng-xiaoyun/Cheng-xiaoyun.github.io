使用hexo搭建博客并上传到github

丑小鸭今天第一次使用hexo搭建博客，记录一下。

前提：已安装git、注册github账号。

  1. 首先要安装node.js，最好安装最新版本。
    博主第一次安装的v4版本，结果后面在使用hexo命令时报一下错误：
    SyntaxError: Unexpected token =at exports.runInThisContext (vm.js:53:16)
    然后根据网友提示升级版本：

    但是并没有成功。
    于是乎卸载重新安装最新版本。

    确认nodejs安装成功就直接在cmd里node –version或在git bash里面输入node回车（命令行变成一个右箭头就是成功。ctrl+c退出）

  2. 安装hexo。
    （1） 利用 npm 命令即可安装。（在任意位置点击鼠标右键，选择 Git bash ）

    npm install -g hexo-cli 或者 npm install -g hexo

    （2）创建hexo文件夹：
    随便在你指定的文件夹下（如 D:\hexo ），(在 D:\hexo内点击鼠标右键，选择 Git bash )，Hexo 即会自动在目标文件夹建立网站所需要的所有文件。
    （3）hexo init
    （4）npm install
    （5） 启动本地服务。现在已经搭建起本地的hexo博客了，执行以下命令(在 D:\hexo )。①hexo generate（或者hexo g）②hexo server（或者 hero s）
    然后到浏览器输入 localhost:4000 看看（如果无法访问，请查看你的4000端口是否被占用）。
    至此，本地博客已经搭建起来了，只是本地的，接下来关联GitHub。

  3. 在github中新建一个public的仓库。
    名字命名为yourname.github.io。（你的github名字加.github.io）

  4. hexo关联GitHub。
    找到你存放Hexo的文件夹（如 D:\hexo ），在里面找到_config.yml文件，用notepad++打开，直接拖到最后，可以看到。

    改成以下，特别注意，在：的后面是有一个空格的，千万要小心，不然后出错的，其中yourname即在github上面的用户名

    保存后，然后在当前文件夹（ D:\hexo ）打开Git Bash Here
    $ hexo g
    $ hexo d
    如果出现以下异常
    ERROR Deployer not found: git
    尝试输入以下命令：
    $ npm install hexo-deployer-git –save
    再重新执行刚刚的两条命令
    这时候如果弹出一个对话框，输入在guthub上面的用户名和密码即可
    这时候我们就可以在浏览器输入http://yourname.github.io（yourname替换成github上的名称）就可以看到博客已经搭建成功了。

  5. 更改hexo样式
    ① 首先在浏览器https://hexo.io/themes/进入github官方主题界面。里面有非常多的主题，选择一个自己喜欢的，点进去 ，就比如说我选了这个：cyanstyle ,点击去之后就会进入这个主题拥有者的博客。然后找到博主的仓库，进入他的仓库，在他的仓库中搜索这个样式的名称，找到对应的下载地址，我找的是（https://github.com/wizardforcel/hexo-theme-cyanstyle.git）
    ②进入你的Hexo地址，比如我的是（ D:\hexo ）打开Git Bash Here，输入
    git clone https://github.com/wizardforcel/hexo-theme-cyanstyle.git
    然后将下载下来的这个主题放入你的D:\Hexo\themes文件夹中，给他命名（如cyanstyle）
    每个主题样式的目录如下：

    ③更改一下hexo目录（ D:\hexo ）下的配置文件_config.yml，主题名修改一下即可

    ④然后清空一下public缓存：hexo clean
    ⑤然后重新生成datebase：hexo g
    ⑥ hexo s 浏览器打开本地仓库（localhost:4000你会发现本地仓库已经变为自己刚刚更改的主题了 ）
    ⑦然后上传到远程仓库就行了：hexo d 在浏览器输入域名http://yourname.github.io就可以看到已经更改的主题。

  6. 上传你的第一篇博客。
    你可以使用CSDN网上面提供的MarkDown编辑器 http://write.blog.csdn.net/mdeditor (在csdn博客写新文章的时候能切换到这种编辑器)，写好文章后，找到菜单栏的“导出到本地”选项 。 以md格式导出到本地，然后copy该md文件，粘贴到你当初建的博客站点（ D:\hexo ）文件夹下的source\ _posts目录下，一个md文件对应一篇博客文章。

接着在你的文件夹（ D:\hexo ）下右键空白处，选择Git Bash Here，
输入：hexo generate（hexo g），回车，生成静态页面，
再输入：hexo server（hexo s），回车，到localhost:4000预览博客效果，
最后输入：hexo deploy（hexo d），回车，同步到github上去就行了。
